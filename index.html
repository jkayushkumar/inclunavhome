Welcome to INCLUNAV
Making Indoor Navigation Smooth and Accessible
Indoor wayfinding and service accessibility are challenging, especially if you visit a large indoor facility like healthcare centres, transport terminals, museums and shopping complexes for the first time. Information unavailability and poor accessibility in these facilities have brought these challenges to stand as a mainstream problem for indoor spaces. INCLUNAV provides a compact and scalable solution.
  • Annotate as many buildings you want
  • Upload floor plan, tag floor elements
  • Tag information such as rooms, doors, services, washrooms and floor connection like lift,stairs on floor plan
  • Test smoothly using our testing interface
  • Make it live to the users
  • Smooth and accesible indoor navigation
  • Easy localization to help user to find a location inside the building
  • Quick search for rooms and nearby services while navigating
  • Easy mutifloor navigation assitance
  • Mobile App and Web versions available
